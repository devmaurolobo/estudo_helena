🌟 Grau Superlativo

🤔 O que é isso?

Sabe quando você quer dizer que algo é SUPER MEGA HIPER de alguma coisa? Tipo que o sorvete é gostoso DEMAIS? 🍦 É aí que a gente usa o grau superlativo!

Tem dois jeitos de fazer isso:

📝 ANALÍTICO: coloca muito, mais, super antes
🍰 muito gostoso · extremamente rápido
✂️ SINTÉTICO: coloca um final mágico (sufixo)
✨ sufixos: -íssimo, -bilíssimo, -císsimo, -imo, -níssimo
🦄 lindo → lindíssimo

🤔 Mau ou Mal?

🎩 O segredo MÁGICO!

MAU 😈 é o contrário de BOM 😇

MAL 💔 é o contrário de BEM 💚

🪄 Truque que NUNCA falha

Tenta trocar a palavra:

Se BOM couber → escreve MAU 😈

Se BEM couber → escreve MAL 💔

Exemplo: "Ele é um mau aluno" 👉 dá pra trocar por "bom aluno" ✓
"Ele dorme mal" 👉 dá pra trocar por "dorme bem" ✓

📍 Onde / Aonde

🤔 Lugar parado ou movimento?

ONDE 🏠 = lugar onde algo ESTÁ. Sem movimento!

AONDE 🏃 = lugar para o qual VAI. Tem movimento!

🏠 Onde você mora? (= em que lugar)
🏃 Aonde você vai? (= para que lugar)
🪄 Truque infalível

Se o verbo é de movimento (ir, chegar, voltar, viajar) ✈️ → use AONDE

Se o verbo é de "parado" (ser, estar, morar, ficar) 🏠 → use ONDE
